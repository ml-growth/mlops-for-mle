caution

📌 해당 파트는 01. Database 파트의 DB 와 03. Model Registry 파트의 모델을 이용합니다.
📌 DB 를 띄우지 않은 경우 01. Database 파트를 완료하고 DB 가 띄워진 상태에서 진행해주세요.
📌 학습된 모델을 불러올 Model Registry 를 띄우지 않은 경우 03. Model Registry 파트를 완료한 상태에서 진행해주세요.
