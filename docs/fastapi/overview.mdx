# 0) Overview
이번 파트에서는 모델을 서빙하기에 앞서 이에 필요한 API 의 개념에 대해 학습합니다.
실습을 위해서는 Python 을 이용해서 API 를 만들 수 있는 웹 프레임워크인 FastAPI 를 사용합니다.

이번 파트에서 구현할 workflow 는 다음과 같습니다.

<div style={{textAlign: 'center'}}>

![FastAPI workflow](./img/fastapi-1.png)
[그림 5-1] FastAPI Workflow

</div>

Docker 를 이용하여 FastAPI 로 만든 API 서버를 실행하고 Client 가 서버에 Request 를 보냅니다.
Request 를 받은 API 서버는 다시 Client 에게 Response 를 주게 됩니다.
